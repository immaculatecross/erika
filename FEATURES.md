# Milestones

Statuses: `backlog` ΓÁÈ `next` ΓÁÈ `building` ΓÁÈ `done`, plus `folded` (subsumed by a later milestone, named in the row). One milestone = one mission handed to the dispatcher. Version scope: **v0.1 = E-1ΓÇªE-5** (D-5, D-12) ┬╖ **v0.2 = E-7, E-9, E-11, E-6, E-12** in that build order (D-15) ┬╖ **v0.3 = E-16ΓÇªE-23** in ID order (D-16, ratified from RETRO-001) ┬╖ **the Learn era (D-17): v0.4 = E-24ΓÇªE-28 ┬╖ v0.5 = E-29ΓÇªE-33 ┬╖ v0.6 = E-34, E-35 ┬╖ v0.7 = E-36, E-37 ┬╖ v0.8 = E-38, E-39 ┬╖ v0.9 = E-40 ┬╖ v1.0 = E-41**, each version in ID order, every prefix a coherent release. E-1ΓÇªE-15 are listed in original roadmap order; the build order is whatever the version decision names.
