# Milestones

Statuses: `backlog` → `next` → `building` → `done`, plus `folded` (subsumed by a later milestone, named in the row). One milestone = one mission handed to the dispatcher. Version scope: **v0.1 = E-1…E-5** (D-5, D-12) · **v0.2 = E-7, E-9, E-11, E-6, E-12** in that build order (D-15) · **v0.3 = E-16…E-23** in ID order (D-16, ratified from RETRO-001) · **the Learn era (D-17, amended by the plan review 2026-07-23; v0.6+v0.7 merged 2026-07-24, operator-directed): v0.4 = E-24…E-29 · v0.5 = E-30…E-34 · **v0.6 = E-35, E-36, E-37, E-38** ("feature-complete") · v0.7 = E-39 · v0.8 = E-40 · v1.0 = E-41 (v0.9 skipped)**, each version in ID order, every prefix a coherent release. E-1…E-15 are listed in original roadmap order; the build order is whatever the version decision names.
