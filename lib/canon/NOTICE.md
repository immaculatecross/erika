# Canon asset — provenance & license (E-33, D-19)

The reading/listening format draws on a **modest, committed, license-clean**
Italian canon. Every passage in `passages.json` is a short excerpt from a work
that is **in the public domain worldwide** — each author died far more than 70
years ago (the longest post-mortem term in any jurisdiction), so no copyright
subsists in these texts anywhere.

This asset is deliberately **small**: it demonstrates the format at one passage
per CEFR band. It is not a library and is not meant to grow into one here.

D-19 license discipline: **public domain only**, attributed. No copyrighted text,
no CC BY-NC or otherwise restricted source, ever enters this path.

Project Gutenberg (both distribute these works as public domain). The CEFR band is
an authoring judgment of the excerpt's reading difficulty (a proxy for matching a
learner's edge), not a measured level.
